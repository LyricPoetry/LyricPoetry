#### 保温杯 
[2023-01-22]

不是所有给温暖的 都是太阳
那无语的是火炉 淡漠的是蜡烛  冰冷的是钢铁

有谁能伴我们 一生漂流呢
为了寻找那些温暖 有个人在风雪中
竟眺望了 很多很多年



#### 窗外的云

抬头望去

那像是一张忘记刮胡的脸

我看你时 正微笑

你看我时  正眉头紧锁





#### 待续



生活不是舞台的话剧  有时不是让朋友，亲属，甚至父母满意





生活有时不是让谁，熟人，朋友，甚至父母满意，

首先要回归的是自己内心。就像走路，

不管姿势多么优美或别扭，在别人看起来，

如果走路的那个人，是不舒服的，其他还有什么意义呢。

我们在生活中，奋力前行，那不是表演不是show，我们即是主角也是观众.其次才是他人呢。
